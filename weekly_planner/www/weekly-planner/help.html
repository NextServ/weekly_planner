Weekly Planner FAQs

How do I create a week plan?
  1. Go to https://portal.theabbasorchard.com/weekly-planner. Make sure you are logged in.
  2. Click New Planner.
  3. Fill out the new planner form with the following:
      4.   a. Instructor - auto-filled
        b. Student Group - a.k.a. Level
        c. Start Date - Sunday of the week
        d. End Date - auto-filled
        e. Description (optional)
        f. Then click Save.
  4. The newly created weekly plan will then appear as the first plan in the list. It is still incomplete as you will still need to add presentations and students to it. To do that, click Edit at the right-most column of the same row of the new week plan.
